css display 속성에 대해 알아보자!

  • display속성은 화면에 어떻게 보여질 지 결정하는 속성
  • html 태그는 기본적으로 block과 inline 요소 둘 중에 하나를 기본값으로 가짐.

inline

  1. span, a, img태그와 같이 내용물( content )만큼만 공간이 할당됨.
  2. content 만큼 공간을 할당하기 때문에, width, height, padding, margin사용 불가
  3. 줄바꿈 없이 바로 옆에 배치
d_inline1 클래스를 가진 div
d_inline2 클래스를 가진 div

block

  1. div, p, ul, ol 태그와 같이 한 줄을다 차지함
  2. inline 요소와 달리width, height, padding, margin사용 가능
d_block1을 가진 div
d_block2을 가진 div

inline-block

  1. block요소처럼 크기 지정을 하되
  2. inline요소처럼 다른 내용들과 같은 줄을 공유하고 싶은 경우
  3. block과 inline 요소의 장점을 가짐
d-iblock1을 가진 div
d-iblock2을 가진 div
